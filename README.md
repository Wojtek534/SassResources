# Style Guide Sass sample
## This is a simple structure for CSS manage using SASS and 7-1 rule
#### Requirements
Node
https://nodejs.org/
NPM
https://www.npmjs.com
Yarn
https://yarnpkg.com
Ruby & Sass
http://sass-lang.com
#### Recomendation
Visual Studio Code
with extensions (Vue 2, Beautify, Beautfiy Css/Sass/Scss, Color highlight)
#### Install dependencies
```
$yarn
OR
$npm install
```
#### Run app to check
```
$npm start
```
Default port = 3000
